# Province

Kami menjangkau seluruh provinsi di indonesia, namun untuk kebutuhan expedisi akan berbeda mengikuti kebutuhan masing-masing

::code-group
```php [PHP]
KiriminAja::getProvince();
```
```bash [POST]
https://tdev.kiriminaja.com/api/mitra/province
```
::
